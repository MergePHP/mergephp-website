New MySQL Features That You May Have Missed

August 13th, 2020

MySQL has gone to a continuous release cycle so that new features are added several times a year. This quickly provides a better product to customers. The news about these new facets may have escaped your notice if you were not paying attention. So you may have missed the addition of hash joins, multi-valued indexes, the ability to validate JSON documents before the are stored, dual passwords, the clone replication tool, and other improvements. This session will cover not only what is new, but how to use those new goodies.

Presented by Dave Stokes
